# 0003 — Drag-drop canvas floor plan

## What to build

Admin can view and arrange tables on a visual canvas. Tables from the list appear as draggable rectangles on a grid. A toggle switches between canvas view and list view. Canvas position updates are persisted to the database.

## Acceptance criteria

- [ ] Canvas view showing tables as positioned rectangles from x, y coordinates
- [ ] Drag to reposition a table, position saved on drop
- [ ] Add new table from canvas (+ button), creates in list
- [ ] Toggle button to switch between canvas and list view
- [ ] Delete table from canvas context menu
- [ ] Responsive — works on tablet and desktop

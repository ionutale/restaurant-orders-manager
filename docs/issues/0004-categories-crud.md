# 0004 — Categories CRUD

## What to build

Admin can create, edit, reorder, and delete menu categories (e.g., Appetizers, Mains, Desserts, Drinks, Wines). Categories have a name, display order, and optional icon. The list view shows all categories with drag-to-reorder.

## Acceptance criteria

- [ ] Database schema for `categories` (id, name, display_order, icon, created_at)
- [ ] Go CRUD endpoints for categories
- [ ] Admin UI page: list with drag reorder, add, edit, delete
- [ ] Mutations recorded in audit_events
